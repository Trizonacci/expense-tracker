# CLI Expense Tracker

A Python command-line application to track expenses with persistent storage.

## Features
- Add and update expenses
- View all recorded expenses
- Calculate total spending
- Persist data between runs using a file
- Handle invalid user input gracefully

## Technologies Used
- Python (standard library only)

## How to Run
1. Clone the repository
2. Run the script:
   python expense_tracker.py

- Structuring a small Python application
- Using functions and dictionaries
- File I/O and data persistence
